# StockProduitApi

## Start Project

### Intall Pipenv

````
pip install pipenv
````

### Install dependencies

````
pipenv install
````

### activate virtualenv
````
pipenv shell
````

### Configure database
Insert your database url in config.py, 
project is configure for postgresSQL.

````
SQLALCHEMY_DATABASE_URI = 'postgresql://name:password@localhost:5432/database'
````

### Run Flask Project

````
flask run
````
